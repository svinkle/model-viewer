Rendering Showcase

  • Many Visible Element

    Demonstrates how <model-viewer> is capable of rendering many simultaneously visible models at once.

  • List of Elements

    Shows how <model-viewer> optimizes rendering only elements that are visible in the viewport.

  • Physically Based Rendering w/ HDR Environments & Radiance Maps

    View models that are rendered with very high quality. WARNING: This demo uses 50MB+ of assets. A local web server or landline internet connection is highly recommended.
